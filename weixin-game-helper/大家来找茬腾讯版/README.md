# 大家来找茬腾讯版

## 功能
- [x] 识别两张图的不同点，标注红色
- [x] 手动点击标红的区域，完成找茬

## 运行
- 安装 Node.js 8.x 以上
- 安装 adb 工具
- 安卓手机连接数据线
- 打开 USB 调试
- 安装依赖 `npm i`
- 运行脚本 `npm start`

成功后，会自动打开一个网页，按照页面提示操作即可。同时请关注终端打印的错误消息。

为了看得更清楚，使用 1:1 的比例显示。如果觉得图片太大可以自行浏览器缩放 `ctrl +/-`
